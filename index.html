Just Go

A travel social media application, where users can share their travel memories with friends.

Just Go was a personal idea that was able to come to life with some help from some classmates. Before I started learning about the intricacies of coding I was writing down ideas of things I wanted to make, this social media travel website was one of them.

Just Go was a project that I was able to lead in designing the interface and collaborating with the user experience. Our goal with the project was to make it simple and fun to use. We didn’t want the site to be too flashy or over the top either because the story that the image was telling was more important.

detail of user post

In addition to creating the design of the app, we were also the ones responsible for taking the designs and creating a functional site. We built it using HTML, CSS, Bootstrap, Vue.js, MongoDB, and Mongoose.

Though I feel like there is always more work that could be done to improve, my team and I were very pleased with the product that we were able to create together!

Keepr

A pinterest-like application where users can share interesting photo with one another.

Keepr is a pinterest-like website where users come to get new ideas and share images that they like. Users can ‘keep’ other user posts in their ‘vaults’ and store them away for when they want to access them.

I worked on this project solo, and again wanted to create a simple color palette and layout as not to distract from the content that users create.

The images that are posted keep a count of how many people viewed it and have kept it into one of their vaults. Users also have the option to share the keep on their Facebook wall.

detail of user post

This project was created using HTML, CSS, Bootstrap, DOTNET, C#.

detail of user dashboard

This was a fun project for me, because I was able to get my feet wet with C# backend coding as well as desgin the application.

Kanban Cloud

A fun task board application, designed to allow users to keep track of all of their to-do’s.

Kanban cloud is a web based personal organizer. This site allows a user to create lists to keep track of tasks that need to be completed.

This project was completed with one other person. We brainstormed ideas and came up with a fun and retro balloon theme. I was able to test my skill with Photoshop and edit some graphics to create the perfect balloon scene for the home page.

to do boards

This project was created with HTML, CSS, Bootstrap, Vue.js, MongoDb, Mongoose. Our goal with this project was to create a simple application that would be aesthetically nice and easy to use. We wanted to solidify our understanding of creating proper relationships between components.

Cuisine Seen

A recipe book app that I started to design. A work in progress, but a joy to desgin and plan out.

Cuisine Seen is an application that I am still in the process of creating. I started out with a problem that I experience in my own life - figuring out what to cook each day. Something seemly simple can be hard to manage when you don't have ample amounts of time on your hands.

With Cuisine Seen, I wanted the application to be a one stop spot for users to be able to create their own cookbooks, add their own or other peoples recipes to their cookbook(s), create a weekly or monthly meal plan, and create a shopping list.

I also wanted to get a good idea of who would be using this application. Probably someone like me? Someone who wants to be able to save time with their meal prep. I determined that the average user would be someone like Kelly here:
